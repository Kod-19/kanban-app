# Kanban App

A modern, responsive kanban board application built with React and Vite for efficient task management and workflow organization.

## Features

- Create and manage kanban boards
- Drag and drop tasks between columns
- Real-time task updates
- Clean and intuitive UI
- Fast development experience with HMR

## Tech Stack

- **React** - UI library
- **Vite** - Build tool and dev server
- **ESLint** - Code quality

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn
